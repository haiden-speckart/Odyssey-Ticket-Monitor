# Odyssey-Ticket-Monitor
# 🎬 Odyssey Ticket Monitor  A serverless Cloudflare Worker that polls Regal Cinemas' API daily over a 45-day window for "The Odyssey" tickets.  ### Features * **Auto-Scan:** Checks 45 days of showtimes via Cron. * **Smart Alerts:** Sends instant push notifications via `ntfy.sh`.
